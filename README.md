## Loop Engineering 橙皮书 / The Loop Engineering Orange Book

**从提示词到闭环工程，AI Agent时代的软件生产方法论**

*A methodology for software production in the AI Agent era — from prompts to closed-loop engineering.*

---

## 关于本书 / About This Book

**中文简介：**

Loop Engineering 是一套全新的软件工程方法论。当 AI Agent 从"辅助工具"升级为"执行主体"，传统的线性开发流程已经失效。本书系统性地提出"闭环工程"（Loop Engineering）框架，教你如何设计可执行、可审核、可学习、可进化的闭环系统，让 AI Agent 真正成为可靠的软件生产力。

**English Introduction:**

Loop Engineering is a groundbreaking software engineering methodology. As AI Agents evolve from "assistive tools" to "execution subjects," traditional linear development workflows break down. This book systematically introduces the "Loop Engineering" framework, teaching you how to design executable, auditable, learnable, and evolvable closed-loop systems — making AI Agents truly reliable software productivity engines.

## 为什么读这本书 / Why Read This Book

- **如果你是工程师** — 你将学会如何从"写代码的人"转型为"设计Loop的人"。
- **如果你是技术管理者** — 你将掌握AI Agent团队的管理方法论和效率度量体系。
- **如果你是创业者** — 你将理解如何用闭环工程构建可持续的技术壁垒。

*If you're an engineer* — Learn to transition from "code writer" to "loop designer."
*If you're a tech leader* — Master the management methodology and efficiency metrics for AI Agent teams.
*If you're a founder* — Understand how to build sustainable technical moats with loop engineering.
